26. 開発ツール¶

この章で紹介されるモジュールはソフトウェアを書くことを支援します。たとえば、 pydoc モジュールはモジュールの内容からドキュメントを生成します。 doctest と unittest モジュールでは、自動的に実行して予想通りの出力が生成されるか確認するユニットテストを書くことができます。 2to3 は Python2.x 用のソースコードを正当な Python 3.x コードに翻訳できます。
