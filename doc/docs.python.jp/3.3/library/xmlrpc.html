21.25. xmlrpc — XMLRPC サーバーとクライアントモジュール¶

XML-RPC は HTTP 経由の XML を使ってリモートプロシージャコールを実現する方法です。 XML-RPC を使うと、クライアントはリモートサーバー (サーバーは URI で名前付けられます) 上のメソッドを引数付きで呼び出して、構造化されたデータを受け取る事ができます。

xmlrpc パッケージは XML-RPC のサーバーとクライアントを実装したモジュールを持っています。モジュール一覧:

  • xmlrpc.client
  • xmlrpc.server
