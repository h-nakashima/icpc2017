18. プロセス間通信とネットワーク¶

この章で解説されるモジュールは他のプロセスと通信するメカニズムを提供します。

Some modules only work for two processes that are on the same machine, e.g. signal and mmap. Other modules support networking protocols that two or more processes can used to communicate across machines.
