31. Python言語サービス¶

PythonにはPython言語を使って作業するときに役に立つモジュールがたくさん提供されています。これらのモジュールはトークンの切り出し、パース、構文解析、バイトコードのディスアセンブリおよびその他のさまざまな機能をサポートしています。

これらのモジュールには、次のものが含まれています:

  • 31.1. parser — Python 解析木にアクセスする
    • 31.1.1. ST オブジェクトを作成する
    • 31.1.2. ST オブジェクトを変換する
    • 31.1.3. ST オブジェクトに対する問い合わせ
    • 31.1.4. 例外とエラー処理
    • 31.1.5. ST オブジェクト
    • 31.1.6. 例: compile() のエミュレーション
  • 31.2. ast — 抽象構文木
    • 31.2.1. Node クラス
    • 31.2.2. 抽象文法 (Abstract Grammar)
    • 31.2.3. ast ヘルパー
  • 31.3. symtable — コンパイラの記号表へのアクセス
    • 31.3.1. 記号表の生成
    • 31.3.2. 記号表の検査
  • 31.4. symbol — Python 解析木と共に使われる定数
  • 31.5. token — Python 解析木と共に使われる定数
  • 31.6. keyword — Python キーワードチェック
  • 31.7. tokenize — Pythonソースのためのトークナイザ
    • 31.7.1. 入力のトークナイズ
    • 31.7.2. コマンドラインからの使用
    • 31.7.3. 例
  • 31.8. tabnanny — あいまいなインデントの検出
  • 31.9. pyclbr — Python クラスブラウザサポート
    • 31.9.1. Class オブジェクト
    • 31.9.2. Function オブジェクト
  • 31.10. py_compile — Python ソースファイルのコンパイル
  • 31.11. compileall — Python ライブラリをバイトコンパイル
    • 31.11.1. コマンドラインでの使用
    • 31.11.2. パブリックな関数
  • 31.12. dis — Python バイトコードの逆アセンブラ
    • 31.12.1. Python バイトコード命令
  • 31.13. pickletools — pickle 開発者のためのツール群
    • 31.13.1. コマンドラインの使い方
      • 31.13.1.1. コマンドラインオプション
    • 31.13.2. プログラミングインターフェース
